== Description ==
Bulk edit variation prices across many products. Supports percent or fixed-amount increases, preview mode, background processing and undo.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/woo-bulk-variation-pricer` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to `Products -> Bulk Variation Price Editor`.
